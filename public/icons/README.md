# PWA Icons

This directory contains the icons for the EMERGE Intervention Planner PWA.

## Required Icons

The following icons are needed for full PWA support:

- `icon-192.png` - 192x192px PNG icon (required for Android)
- `icon-512.png` - 512x512px PNG icon (required for Android)
- `apple-touch-icon.png` - 180x180px PNG icon (recommended for iOS)

## Temporary Placeholder

A placeholder SVG (`placeholder.svg`) is included for development.

## Creating Icons

To create the required PNG icons from the placeholder SVG or your own design:

1. Create a high-quality 512x512px design
2. Export as PNG at the following sizes:
   - 192x192px → `icon-192.png`
   - 512x512px → `icon-512.png`
   - 180x180px → `apple-touch-icon.png`

Alternatively, use an online tool like:
- https://realfavicongenerator.net/
- https://favicon.io/

## Design Guidelines

- Use the brand colors: `#FF006E` (primary) and `#FFFDF5` (background)
- Ensure the icon is clear and recognizable at small sizes
- Consider using the letter "E" or a book/education symbol
- Maintain consistency with the app's visual identity
